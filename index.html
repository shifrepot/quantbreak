Link Violation — Why SDR (SIR) Instead of OLS

The true model is Y = f(β₀ᵀX) where f is nonlinear. Black-Scholes imposes a linear, log-normal structure — this is a textbook Link Violation. Even under Link Violation, if E[X|β₀ᵀX] is linear (Linear Conditional Mean), the OLS direction β̂ still recovers β₀ up to scale. We can find the true risk direction without knowing f.

Y = f(β₀ᵀX) + ε         // True model: unknown nonlinear f (Link Violation)
E[X | β₀ᵀX] = Σ·η(β₀ᵀX)  // Linear Conditional Mean holds ← key assumption (L9)
β̂_OLS ∝ β₀             // ∴ direction recoverable even when model is wrong

MODULE C · REGIME-MIXTURE CVaR vs BLACK-SCHOLES CVaR

Black-Scholes computes CVaR from a single log-normal distribution — ignoring regime structure entirely. Using regime weights from the Born Rule measurement above, we compute a Regime-Weighted CVaR: a weighted average of per-regime tail losses. This is not "quantum math vs classical math." It is single-state assumption vs multi-regime reality. QAOA then minimizes this regime-aware CVaR over the leverage space.

WHY QAOA?
Not for speed. At this demo scale (5 candidates), classical search is sufficient.

QAOA is used because Regime-Weighted CVaR (from SIR regimes) naturally maps to a QUBO — the same structure QAOA is designed to optimize. As asset count and constraints grow, the combination space expands exponentially and classical solvers face increasing difficulty.

This project does not claim quantum speedup. The objective is to formulate regime-aware leverage selection within a quantum optimization framework.
